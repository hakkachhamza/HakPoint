HakPoint is a modern Enterprise Management Platform focused on performance, security, scalability, and clean architecture.

The goal of this project is to provide developers and businesses with a reliable foundation for building powerful management systems while encouraging collaboration within the open-source community.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/HakPoint.git
```

## Navigate

```bash
cd HakPoint
```

## Install Dependencies

```bash
composer install
```

## Configure Environment

Copy

```text
.env.example
```

Rename it to

```text
.env
```

Configure your database credentials.

---

## Import Database

Import

```text
install.sql
```

using phpMyAdmin or MySQL Workbench.

---

## Launch

Start your local PHP server.

Example:

```bash
php -S localhost:8000
```

Open

```
http://localhost:8000
```
